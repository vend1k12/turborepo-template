# Shared Packages Directory

This section provides API definitions and integration guides for all shared packages located under the `packages/` workspace root.

## Package List

1. **[UI Package (@vend1k/ui)](./ui.md)**
   Central design tokens, Tailwind v4 setups, and shared Shadcn components.
2. **[Shared Package (@vend1k/shared)](./shared.md)**
   Browser-safe shared schemas, constants, user permissions, and types.
3. **[Database Client (@vend1k/db)](./db.md)**
   PostgreSQL connection pools and Drizzle ORM schemas.
4. **[Authentication Engine (@vend1k/auth)](./auth.md)**
   Better Auth engine config and client-side auth helpers.
5. **[API Client Factory (@vend1k/api-client)](./api-client.md)**
   Typed Hono RPC client initialization.
6. **[Shared Configs](./config.md)**
   Unified configurations for typescript-config and eslint-config.
