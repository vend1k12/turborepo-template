# Applications Directory

This section provides operational profiles and interface definitions for all service interfaces inside the `apps/` directory.

## App List

1. **[Web Frontend (apps/web)](./web.md)**
   Public landing site, marketing features, pricing grids, and brand introduction layout.
2. **[Docs Portal (apps/docs)](./docs.md)**
   Technical documentation console rendered dynamically from system pages and guides.
3. **[User Portal (apps/user)](./user.md)**
   Customer workspace dashboard displaying personal settings, usage analytics, profile actions, and organization membership.
4. **[Admin Console (apps/admin)](./admin.md)**
   Internal operations dashboard for auditing users, executing database edits, and managing UI themes.
5. **[Hono Backend API (apps/api)](./api.md)**
   The core TypeScript routing server containing Better Auth controllers, database connections, schemas, and routes.
