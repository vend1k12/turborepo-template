The **vend1k personal** starter is a Turborepo-powered monorepo combining a [Hono](https://hono.dev) backend API with multiple [Next.js](https://nextjs.org) frontends, wired together with end-to-end type safety, a shared design system, and credential-based authentication.

These docs cover how the workspace is organized, how the pieces fit together, and how to extend them.
