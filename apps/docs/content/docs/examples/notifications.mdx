A toast and persistent drawer notification center. It polls endpoints periodically or subscribes to SSE streams to alert the user of account updates.

## Notification types

- **Toast alerts:** immediate micro-feedback on user actions.
- **In-app alerts:** persistent items that can be marked read/unread.

Toasts are ephemeral and dismiss themselves, while in-app alerts accumulate in a drawer so the user never misses an important update.
