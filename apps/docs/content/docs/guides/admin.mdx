---
title: Admin Panel Console
description: The internal administration console.
---

The admin workspace is a feature-rich console styled with the shared `@vend1k/ui` components. It enables administration over users, logs, and database records.

## Key capabilities

- **User auditing:** view, edit, suspend, and search user accounts via server-paginated tables.
- **Theme controls:** custom themes are selectable on the fly through the shared theming system.
- **Analytics:** embedded charts for tracking user signups and engagement rates.

Like the user portal, the admin console is built on the shared dashboard shell, so the two apps differ in capability rather than in structure.
