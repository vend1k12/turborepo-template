The User Portal (`apps/user`) is a standalone application that gives customers or team members access to their credentials, profile details, and organization parameters.

## Available routes

- `/sign-in` and `/sign-up` — authentication screens with form validation.
- `/dashboard` — high-level overview of active sessions and recent account events.
- `/profile` — edit contact details and update account settings.
- `/organization` — manage workspaces, roles, and invite colleagues.

The portal shares the `@vend1k/ui` dashboard shell with the admin console, so navigation, theming, and layout stay consistent across both apps.
